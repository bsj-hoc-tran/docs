# 📊 ストーリーポイント管理システム

モダンで使いやすいインターフェースを持つ作業見積もりポイント管理システムです。

## ✨ 機能

### 🎨 モダンなインターフェース
- **レスポンシブデザイン**: すべてのデバイス（デスクトップ、タブレット、モバイル）で快適に動作
- **鮮やかなカラー**: 紫と青のグラデーションを基調とした美しいデザイン
- **スムーズなアニメーション**: ホバー、クリック、タブ切り替え時のなめらかなエフェクト
- **ダークモード対応**: ダークモードへの拡張が可能

### 📑 タブによる分類
- **📋 依頼作業**: インフラとサーバーの依頼タスク
- **🔄 定型タスク**: リリース、テスト、レビュー、セキュリティチェック
- **📦 その他タスク**: 環境構築、設定
- **🐛 改善・バグ系**: 難易度と対応範囲による分類ガイドライン
- **🖥️ サーバー**: API、データベース、バッチ処理の詳細ポイント
- **💻 Webフロント**: ページ、UI、API連携、バリデーション
- **📱 アプリ**: UserData、Personal、API、UI作成
- **☁️ インフラ**: AWS、CI/CD、Terraform
- **📐 設計**: 内容確認と詳細設計

### 🔍 スマート検索
- テーブル内のリアルタイム検索
- 入力に応じて自動的に結果をフィルタリング
- 日本語検索対応

### 🔢 テーブルソート
- ヘッダーをクリックしてソート
- 数字と文字列のソートをサポート
- 昇順/降順の切り替え

### ⌨️ キーボードショートカット
- **Alt + 1-9**: タブ間の素早い切り替え
- ナビゲーション時間の節約

### 💾 状態の保存
- 最後に選択したタブを自動的に記憶
- ページリロード時も位置を保持

### 🎯 バッジとタグ
- **難易度**: 低（緑）、中（黄）、高（赤）
- **対応範囲**: 低、中、高
- **領域**: インフラ、サーバー、フロントエンド、アプリ
- **ストーリーポイント**: グラデーションで強調表示

### 📊 カードレイアウト
- サーバー、フロントエンド、アプリ、インフラのセクションはカード形式
- 情報の比較と検索が容易
- 鮮やかなホバーエフェクト

## 🚀 使用方法

### ファイルを開く
Webブラウザで `index.html` ファイルを開くだけです：
```bash
open index.html
# または
firefox index.html
# または
chrome index.html
```

### ナビゲーション
1. **タブ切り替え**: 上部のタブボタンをクリック
2. **検索**: 検索ボックスにキーワードを入力（利用可能な場合）
3. **ソート**: テーブルのヘッダーをクリックしてソート
4. **ショートカット**: Alt + 数字でタブを素早く切り替え

## 🔧 カスタマイズ

### 色の変更
`resources/style.css` のCSS変数を編集：
```css
:root {
    --primary-color: #4f46e5;
    --secondary-color: #06b6d4;
    /* ... */
}
```

### 新しいタブの追加
1. `.tabs` にボタンを追加：
```html
<button class="tab-button" data-tab="new-tab">🆕 新しいタブ</button>
```

2. 対応するコンテンツを追加：
```html
<div id="new-tab" class="tab-content">
    <h2>🆕 新しいタブ</h2>
    <!-- コンテンツ -->
</div>
```

### 新しいデータの追加
既存のテーブルまたはカード構造をコピーしてデータを更新します。
